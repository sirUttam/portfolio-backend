# FastAPI Portfolio Backend

A backend system built using FastAPI for portfolio and learning backend development for ML/AI systems.

## Features
- JWT Authentication (Login system)
- Admin-only access
- CRUD operations
- Database integration
- Secure API routes

## Tech Stack
- FastAPI
- Python
- SQLAlchemy
- SQLite
- Uvicorn
- REST
